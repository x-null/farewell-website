Masterserver

xNULL GameSpy replacement

We had a custom masterserver running before GameSpy shutdown. Initially it didn't support GameSpy protocol and it was used to feed our online server browser.

Later on we've added GameSpy protocol support. This work has been crucial in keeping the game alive.
